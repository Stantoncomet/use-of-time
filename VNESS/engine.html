Welcome to a Visual Novel Engine for Simple Stories, aka VNESS.

This is a story telling medium. This does not contain a game by itself. But you are the player.

Click SELECT GAME FOLDER to select the folder in the same directory as this engine.html file, containg your game data.

A game is broken into paths, each folder is a scene, containg the scene.png, along with folders label numerically, one for each path that branches off of that scene.

Type numbers on your keyboard or press the buttons to select the options that appear.
